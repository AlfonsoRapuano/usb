Advance Product Bundle Pack with Print Option in Invoice Odoo App

Product bundle pack advance odoo app helps user to create product bundle and add different products as a pack, there is different options to manage stock for product bundle pack, and different options for invoice for product bundle pack. Product bundle Pack Odoo apps is offering several products for sale as one combined product which you can able to sell as combo product kit easily from sales order and able to deliver all products pack of kit in delivery orders. It is a common feature in many imperfectly competitive product markets where price plays important roles, using these odoo module you can act set competitive price for same or different products and variants to increase your sales graph.

Features

Sell Product Bundle Pack.

User can create and sell product bundle pack.

Check Stock on Product Pack.

User can check and manage stock of product bundle pack.

Product Bundle Pack Stock Management.

User can manage product bundle pack and product bundle pack items stock using different options.

Product Bundle Pack Invoice Options

User can see and view product on invoice line using different invoice options.

Add Product and Variants in Bundle.

User can add multiple products and products variants with different price for different product pack.

Separate Seller's Product Collection and Management/

User can create separate seller's product collections and management. User can add direct discount on product pack.

Product Bundle Pack Filter.

User can view product bundle pack using "Is Product Pack" filter on product.

Bundle Multiple Instances of the Same Product.

Boost users up-selling performance by offering kit versions of popular products paired with one or more related items and offers bulk discounts.

Product Bundle Pack Price Calculation.

User can calculate bundle price based on added products price on pack or define separate price.

Print Invoice Report based on Invoice Option.

User can print and view product bundle pack on invoice report according to selected invoice option.

Product Bundle Pack Menu

User can create and view product bundle pack from Sales > Products > Product Pack menu.

Product Pack Price

User can add price for product bundle pack, User can also calculate product bundle price from bundle items by enabling "Calculate Pack Price" option.

User can not create variant for product bundle packs.

User can add multiple product items to product bundle pack.

User can apply product bundle pack filter in product kanban view.

Product Bundle Pack Configurations

Under Sales > Configuration > Settings menu user have to select invoice and stock management options for pack items.

If user selects "Only main Bundle product Pack" as invoice option then only product bundle pack will available on invoice lines, and user selects "Deduct Stock Of Only Bundle Items" option as stock management then only product pack items quantity will deducted from stock.

Available On Hand Quantity

User can see available onhand quantity for product and product pack.

User can add whole bundle in sale order.

User can see delivery order with selected options.

Updated On Hand Quantity

User can see updated on hand quantity for product and product pack.

User can see created invoice with selected options.

User can see printed invoice report with invoice option 1: "Only main Bundle product Pack".

Product Bundle Pack Configurations

If user selects "Only Bundle Pack On Invoice Lines" as invoice option then only product bundle pack item will available on invoice lines.

User can see created sales order with selected options.

User can see available product pack items on invoice lines.

User can see printed invoice report with invoice option 2: "Only Bundle Pack on Invoice Line".

Product Bundle Pack Configurations

If user selects "Main Bundle With Pack Item On Invoice Line" as invoice option then product bundle pack with product pack items will available on invoice lines, and user selects "Deduct Stock of Main Bundle and Bundle Items Both" option as stock management options then product bundle pack and it's items quantity will deducted from stock.<

Available On Hand Quantity

User can see created sales order.

User can see created delivery order with selected options.

Updated On Hand Quantity

User can see updated on hand quantity for product and product pack.

User can see created invoice with selected options.

User can see printed invoice report with invoice option 3: "Main Bundle With Pack Item On Invoice Line".
